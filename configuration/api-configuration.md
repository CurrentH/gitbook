# API Service

`PORT=1234`

API service port

`PAGE_ITEMS_LIMIT=10`

Default number of items to serve for runs feed

`APOLLO_PLAYGROUND=false`

Enable or disable Apollo playground landing page

`BASE_PATH="/graphql"`

Service's base path, useful for reverse proxies
